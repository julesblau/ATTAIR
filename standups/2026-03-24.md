## 🧠 Decisions Made

1. **CORS not changed** — Security flags it as CRITICAL but PM rules prohibit CORS changes without live testing. Infrastructure is ready (`CORS_ORIGINS` env var). Dedicated session recommended.
2. **Light mode not removed** — Has 28+ conditional CSS entries; threshold for removal (< 20) not met. Left intact per task instructions. If you want dark-only, that's a deliberate PM call.
3. **Occasion: silent null, not 400** — Old app versions may send unknown occasion values. Hard-rejecting would break them. Silent null-out is conservative and safe.
4. **Pairings tracked_url returned** — `/api/suggest-pairings/track-click` now returns a `tracked_url` routing through `/api/go/:clickId`. When we onboard affiliate partners, pairing clicks are already tracked.
5. **`t` variable shadowing fixed immediately** — Caught during E2E review. Renamed to `tierKey` to keep i18n `t()` function unambiguous.
