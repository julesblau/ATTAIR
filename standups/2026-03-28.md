### Decisions Made
- **CLI args for agents:** Passed the full AGENTS JSON object via `--agents` CLI flag rather than writing to `.claude/agents/` directory files. This keeps agent definitions co-located in `run.js` and avoids creating new project structure.
- **Stream-JSON parsing:** Used Node.js `readline.createInterface()` for line-by-line JSON parsing of `--output-format stream-json` output, matching the event types the SDK was providing.
- **Rate limit handling:** Preserved the same retry-with-sleep logic, now also checking stderr for rate limit messages in addition to stream events.
- **Committed pre-existing discord-bot changes:** The image attachment support was already written but uncommitted — included it in this commit since it was a clean, working change.
